У цьому завданні вам потрібно буде реалізувати принцип Expert для надання відповідальності належним об'єктам.

1.Створіть класи:

a. Створіть клас User, який представляє користувача системи.

b. Створіть клас Address, що містить інформацію про адресу користувача.

2.Налаштуйте відповідальність:

a. Надайте класу User відповідальність за зберігання адреси користувача, використовуючи об'єкт класу Address.

b. Клас User буде мати методи для роботи з адресою, такі як setAddress() та getAddress().

3.Створіть клас для демонстрації:

a. Створіть клас Main, який містить метод main().

b. У методі main() створіть об'єкт класу User та об'єкт класу Address.

c. Встановіть адресу для об'єкта користувача за допомогою методу setAddress().

d. Виведіть адресу користувача на консоль, використовуючи метод getAddress().

4.Перевірте правильність роботи:

a. Переконайтеся, що адреса користувача належить об'єкту класу User.

b. Переконайтеся, що клас User відповідає за доступ до адреси та виконує необхідні дії з нею.

5.Залийте виконаний проєкт у свій GitHub репозиторій, посилання на який зазначте в LMS
